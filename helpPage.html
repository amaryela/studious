What is Studious?

Studious is an online application that provides both fun and education. It's a Web application that allows you to take a quiz in real time, create your own quiz, and view your scores in a leaderboard.

Quiz

How can I make a quiz?

To create a quiz you need to:
  • Create an account as professor.
  • Login to your Studious account as professor.
  • Click create a room button, make a quiz and setup the quiz by choosing a type of questionaire and set a date and time when the quiz would begin.

Can I make a second quiz start right away after a first quiz is finished?

Yes, if the quiz host schedules a two quizzes on the same date but at separate time, the second quiz can begin right away after taking the first quiz. Note:
  • It is not possible to take two quizzes at once.

Is it possible to students to take a late quiz?

If you're late and merely want to take the quiz, you can catch up until the timer runs out, but you won't be able to go back to the previous quest To avoid being late, a notification announcing that the quiz is about to begin would be displayed before the quiz began or the quiz host would remind them.

How can a missing quiz can be submitted?

Currently there is no way to retrieve a missing quiz, and get back to "Join a quiz" state. Perhaps the best solution is to plan ahead. Alternatively, yo
    1. Invite the students, remind them that there is a quiz given in the room. 2. Allow them to start a new attempt.

What code should I enter to join a room?

To join a room you need to enter a code provided by your quiz host or instructor. You cannot join until you receive a code from your quiz host,you may contact your quiz host to be able to join.
